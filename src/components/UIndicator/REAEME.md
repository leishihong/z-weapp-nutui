## API

### Props

| 参数     | 说明                                                                  | 类型    | 默认值 |
| -------- | --------------------------------------------------------------------- | ------- | ------ |
| current  | 当前步骤                                                              | Number  | 1      |
| size     | 步骤长度                                                              | Number  | 3      |
| block    | 是否启用块级布局                                                      | Boolean | false  |
| align    | 对齐方式，仅在 block 为 true 时生效, 可选值 'left', 'right', 'center' | String  | left   |
| vertical | 是否竖向展示                                                          | Boolean | false  |
